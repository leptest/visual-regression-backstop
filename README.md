### Getting started

Install the packages:
```
npm install
```

Set up reference images (already done, but you can run it again):
```
npm run reference
```

Run a new test:
```
npm run test
```

Then you can view the generated report at `.\backstop_data\html_report\index.html`


### Moving forward
```
npm run approve
```

### Useful BackstopJS commands
```
backstop init
backstop reference
backstop test
backstop approve
```
